# vue-shopping

> 一个基于vue2.0全家桶的极客商城项目。
> 支持响应式终端，路由跳转，筛选排序，购物车结算。

### 技术栈

#### 前台
> * vue
> * vue-router
> * vuex
> * axios
> * es6
> * localStorage
> * webpack

#### 后台
> * Node(Express)

### [项目预览](http://www.heartbrokenkid.com/projects/shopping/)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
